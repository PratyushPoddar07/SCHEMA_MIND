# Quick Start Guide

## üéØ Get Started in 5 Minutes

### Step 1: Prerequisites Check
```bash
# Check Docker
docker --version

# Check Docker Compose
docker-compose --version
```

### Step 2: Get Your Anthropic API Key
1. Go to https://console.anthropic.com/
2. Sign up or log in
3. Navigate to API Keys
4. Create a new API key
5. Copy the key (starts with `sk-ant-`)

### Step 3: Clone and Setup
```bash
# Clone the repository
git clone <your-repo-url>
cd querymind-ai

# Run the setup script
./setup.sh
```

### Step 4: Configure Your API Key
Edit the `.env` file:
```bash
nano .env  # or use your preferred editor
```

Replace `your_anthropic_api_key_here` with your actual key:
```env
ANTHROPIC_API_KEY=sk-ant-api03-xxxxx...
SECRET_KEY=<auto-generated>
```

### Step 5: Start the Application
```bash
docker-compose up -d
```

### Step 6: Access the Application
Open your browser and go to:
- **Frontend**: http://localhost:5173
- **API Docs**: http://localhost:8000/api/docs

### Option 1: Use the Sample Database (Recommended for Testing)

We'll create a sample e-commerce database:

```bash
# Connect to PostgreSQL
docker-compose exec postgres psql -U querymind -d querymind

# Create sample tables
CREATE TABLE customers (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100),
    created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE orders (
    id SERIAL PRIMARY KEY,
    customer_id INTEGER REFERENCES customers(id),
    total_amount DECIMAL(10, 2),
    status VARCHAR(20),
    order_date TIMESTAMP DEFAULT NOW()
);

CREATE TABLE products (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    category VARCHAR(50),
    price DECIMAL(10, 2),
    stock INTEGER
);

CREATE TABLE order_items (
    id SERIAL PRIMARY KEY,
    order_id INTEGER REFERENCES orders(id),
    product_id INTEGER REFERENCES products(id),
    quantity INTEGER,
    unit_price DECIMAL(10, 2)
);

-- Insert sample data
INSERT INTO customers (name, email) VALUES
    ('John Doe', 'john@example.com'),
    ('Jane Smith', 'jane@example.com'),
    ('Bob Johnson', 'bob@example.com');

    ('Mouse', 'Electronics', 29.99, 200),
    ('Desk Chair', 'Furniture', 199.99, 30),

INSERT INTO orders (customer_id, total_amount, status) VALUES
    (1, 1029.98, 'completed'),
```

### Option 2: Connect Your Existing Database

In the QueryMind UI:
1. Click "Add Database" in the sidebar
2. Enter:
   - **Name**: My Database
   - **Type**: PostgreSQL (or your database type)
   - **Connection String**: 
     ```
     postgresql://username:password@host:port/database
     ```
3. Click "Test Connection"
4. Click "Save"

## üîß Troubleshooting

### Services won't start
```bash
# Check logs
docker-compose logs

# Restart services
docker-compose restart
```

### Can't connect to database
```bash
# Check if PostgreSQL is running
docker-compose ps postgres

# Test connection
docker-compose exec postgres psql -U querymind -d querymind -c "SELECT 1"
```

### API errors
```bash
# Check backend logs
docker-compose logs backend

# Verify API key is set
docker-compose exec backend env | grep ANTHROPIC
```

### Frontend not loading
```bash
# Check frontend logs
docker-compose logs frontend

# Rebuild frontend
docker-compose up -d --build frontend
```
